# 04 単体テスト仕様書

## 1. テスト方針

### 1.1 対象範囲
- バックエンド: Rustの各モジュール（ユーティリティ・サービス層・ハンドラ層）
- フロントエンド: Vueコンポーネント・Piniaストア・composable関数

### 1.2 テストフレームワーク

| 層 | フレームワーク |
|----|-------------|
| Rust | `cargo test`（標準） |
| Vue/TypeScript | Vitest + Vue Test Utils |

### 1.3 カバレッジ目標
- バックエンド: 行カバレッジ 80% 以上
- フロントエンド: 行カバレッジ 70% 以上
