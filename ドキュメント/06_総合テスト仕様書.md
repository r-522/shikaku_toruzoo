# 06 総合テスト仕様書

## 1. テスト方針

### 1.1 目的
Cloud Run にデプロイされた本番相当環境で、エンドツーエンドの動作・セキュリティ・パフォーマンスを検証する。

### 1.2 テスト環境
- URL: Cloud Run デプロイ済みサービス
- ブラウザ: Chrome 最新版（DevTools 使用）
- 使用ツール: curl, ブラウザ, gcloud CLI
